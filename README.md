# JavaDoc Generator

A small browser-based JavaDoc builder for Java source comments.

## What it does

- Generates JavaDoc for classes, interfaces, enums, methods, constructors, and fields.
- Supports package declarations and import blocks.
- Supports copyright headers and common open-source license templates.
- Provides documentation profiles for different source-style conventions.

## Documentation profiles

- Strict standard Javadoc: the neutral, recommended profile.
- Oracle / OpenJDK-like: metadata-friendly output that resembles JDK source comments.
- Google Java Style-like: concise JavaDoc with less metadata.
- Open Source Project preset: enables a publication-ready header and uses the neutral profile defaults.

## Run locally

Open `index.html` in a browser.
